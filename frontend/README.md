# NaijaTransit Frontend

This directory contains all frontend files for the NaijaTransit platform.

## Files
- `index.html` - Main user interface
- `admin.html` - Admin dashboard
- `style.css` - Main stylesheet (purple theme)
- `script.js` - Main JavaScript functionality
- `admin-style.css` - Admin-specific styles
- `admin-script.js` - Admin-specific JavaScript

## Assets
- `images/` - All image assets
- `assets/` - Icons, fonts, and additional JavaScript libraries

## Usage
Open `index.html` in a web browser or deploy with a web server.

For development with live reload, use:
```bash
python -m http.server 8000
