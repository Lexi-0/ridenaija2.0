# Ridenaija - Road Transport Booking Platform

A full-stack web application for booking affordable road trips across Nigeria.

## Features
- User authentication and authorization
- Search and compare trips by route and price
- Book trips with seat selection
- Paystack payment integration
- Email notifications
- Admin dashboard for trip management
- Responsive design with purple accent theme

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Python Flask
- **Database**: PostgreSQL
- **Payment**: Paystack API
- **Email**: Flask-Mail with SMTP
- **Deployment**: Docker, Nginx

## Quick Start

### Prerequisites
- Python 3.8+
- PostgreSQL 12+
- Docker (optional)

### Installation
1. Clone the repository
2. Set up environment variables
3. Install dependencies
4. Initialize database
5. Run the application

See [SETUP_GUIDE.md](docs/SETUP_GUIDE.md) for detailed instructions.
